# ProofOrigin Evidence Layer

This directory stores forensic evidence records generated by ProofOrigin AI.

Each scan may contain:

- Report ID
- Timestamp
- SHA-256 image hash
- Metadata snapshot
- AI probability score
- Consensus intelligence results
- Provenance indicators
- Visual forensic findings
- Engine outputs
- User feedback
- Authenticity classifications

Purpose:
Build a permanent forensic intelligence and provenance system for digital media authenticity research.

Future integrations:
- Sightengine
- OpenAI Vision
- OpenAI Reasoning
- Provenance graphing
- Human feedback learning
- Adversarial detection
- Cross-engine consensus scoring
